About This Application

Static GitHub Pages recreation

CDC Topcon Models - Comprehensive Results Explorer

This GitHub Pages application provides interactive exploration of the March 27, 2026 Topcon refresh glaucoma model results without requiring a Shiny server.

Key Features
  • Summary dashboard with top-performing model snapshots.
  • Interactive ROC curves, forest plots, and threshold explorer views.
  • Downloadable tables and filtered CSV exports.
  • Client-side rendering from precomputed analysis outputs only.
Models Evaluated
  • Unimodal: Fundus
  • Multimodal: RNFL+Circle, RNFL+Circle+Fundus
Outcomes Analyzed
  • Glaucoma+Suspect vs Healthy (training definition)
  • Glaucoma vs Suspect+Healthy (stricter definition)
