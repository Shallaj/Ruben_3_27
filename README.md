# Ruben_3_27

GitHub Pages repository for the CDC Topcon Models Explorer static site.

The published site lives in [`docs/`](docs).

To serve it with GitHub Pages:

1. Open the repository settings.
2. Go to **Pages**.
3. Set the source to **Deploy from a branch**.
4. Choose branch `main` and folder `/docs`.

Local source project:

- `HGC_SH/projects/3029_Ruben_CDC_Topcon_3_27_2026/docs`
